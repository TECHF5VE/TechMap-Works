3.第三题答案
>在Python3以前的版本中，整数除以整数，结果一定是整数。但是在Python3及以上版本中，软件系统会智能化地将整数转化成为浮点数来进行运算
>7/3结果为浮点数，7//3表示整除，结果是整数2
>7.0//3与7//3结果在大小上相同，但其中一个为浮点数的时候结果为浮点数，即2.0
